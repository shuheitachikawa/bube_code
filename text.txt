今回は、Web制作分野でオススメの、VScodeの拡張を7つ紹介します。

1. Prettier (プリティア)
フロントエンド開発必須のコードフォーマッター。
インデントのスペース数、改行位置など、コーディングに関するルールをJSONで定義すると、ファイルを保存時に自動でコードを整形してくれます。

2. Auto Rename Tag
HTMLのタグの名前を変えると、終了タグも同時に変更してくれるものです。
地味に、めちゃくちゃ便利で必須です。

3. Code Spell Checker
英単語のスペルチェックをしてくれます。
単語のスペルミスは結構恥ずかしいので、入れておくことをお勧めします。
単語のスペルが曖昧なときにも、合ってるとアラートが消えるので役に立ちます。

4. GitLens
Gitと連動して、コード中にGitのさまざまな情報を表示してくれます。
特に、コードにカーソルを当てると「誰がそのコードを書いたか」を即座に確認することができ、とても便利で必須です。

5. Color Highlight
カラーコードを書くと、その色の枠でコードをハイライトしてくれます。
ハイライトの方法も、枠線 or 塗りつぶし など選択できます。
書いたカラーコードが、正しいかのチェックにもなるので、Web制作やフロントエンド開発には必須です。

6. Live Server
ローカルサーバーを起動して、書いているHTMLファイル表示してくれます。
LiveServerで起動すると、画面リロードせずとも自動でコードの内容を反映してくれるようになります。
Web制作では、あるとかなり便利です。

7. GitHub Copilot
有料ツール(10ドル/月)ですが、AIでコードを補完してくれたり、もはや全て書いてくれるツールです。
あまりにも優秀なので、初学のうちは使わない方が学習が捗るかもです。笑
